Download OpenSIPS v1.5
Next

(:toc-float Table of Content:)

There are multiple options when it comes to downloading OpenSIPS 1.5 :

Release files from WEB site

Tarballs with sources of the 1.5.3 release may be downloaded directly from the project web site: 

      http://opensips.org/pub/opensips/1.5.3/src/opensips-1.6.4-notls_src.tar.gz 
      http://opensips.org/pub/opensips/1.5.3/src/opensips-1.6.4-tls_src.tar.gz

Also, we have nightly builds of tar archives, which are extracted from the latest 1.6.x branch. In order to download the nightly build, go to 

      http://opensips.org/pub/opensips/1.5.3/src/

and download the file with the -svn[REV] suffix.

Tarballs from SourceForge

The same release files we have on the website are also hosted on the SourceForge download area - it is advisable to use it as geographical mirrors are available for a faster download: 

      https://sourceforge.net/projects/opensips/files/OpenSIPS/1.5.3/

Packages

Thanks to several maintainers, OpenSIPS packages for certain Operating System/Distributions are available. The version you are looking for may or may not be available, so please check with our repositories.

SVN download

Although the OpenSIPS project has moved to GIT, you can still download a read-only copy from the SourceForge SVN, which is permanently kept in sync with the GIT repo.

To check out the latest 1.5 source code directly from the SVN you can run : 

   # svn co https://svn.code.sf.net/p/opensips/svn/branches/1.5 opensips_1_5

GIT download

GitHUB hosts the main repository for OpenSIPS. In order to checkout the latest version of OpenSIPS 1.6 you can run : 

   # git clone -b 1.5 https://github.com/OpenSIPS/opensips.git opensips_1_5
