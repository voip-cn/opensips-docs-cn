Documentation -> Manuals

The OpenSIPS Manuals contain description of how to download, install and configure OpenSIPS. The configuration docs cover the scripting language (variables, transformations, flags, routes, operators and statements), the modules (functions, parameters) and the OpenSIPS Interfaces.
