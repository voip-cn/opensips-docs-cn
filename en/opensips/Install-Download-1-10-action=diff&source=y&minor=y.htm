There are multiple options when it comes to downloading '''OpenSIPS''' 1.10 :

----
[[#osippub]]
!!!! Release files from WEB site

Tarballs with sources of the 1.10 release may be downloaded directly from the project web site:
[[http://opensips.org/pub/opensips/1.10.0/src/opensips-1.10_src.tar.gz ]]

Also, we have nightly builds of tar archives, which are extracted from the latest 1.10.x branch. In order to download the nightly build, go to

[[ http://opensips.org/pub/opensips/1.10.0/src/ ]]

and download the file with the -svn[REV] suffix.

----
[[#osipsf]]
!!!! Tarballs from SourceForge

The same release files we have on the website are also hosted on the SourceForge download area - it is advisable to use it as geographical mirrors are available for a faster download:
[[https://sourceforge.net/projects/opensips/files/OpenSIPS/1.10.0/]]


----
[[#osipmi]]
!!!! Packages
Thanks to several maintainers, '''OpenSIPS''' packages for certain Operating System/Distributions are available. The version you are looking for may or may not be available, so please check with [[Downloads/Downloads#toc4|our repositories]].

----
[[#svn]]
!!!! SVN download

Although the OpenSIPS project has moved to GIT, you can still download a read-only copy from the SourceForge SVN, which is permanently kept in sync with the GIT repo.

To check out the latest 1.10 source code directly from the SVN you can run :

[@# svn co https://svn.code.sf.net/p/opensips/svn/branches/1.10 opensips_1_10 @]

----
[[#git]]
!!!! GIT download

GitHUB hosts the main repository for OpenSIPS. In order to checkout the latest version of OpenSIPS 1.10 you can run :

[@# git clone -b 1.10 https://github.com/OpenSIPS/opensips.git opensips_1_10 @]
